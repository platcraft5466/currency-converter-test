## Overview

This project implements a currency converter API that reads conversion rates from CSV file. The API is built with TypeScript, and desingned to run locally for development and on CludeFare workers for production deployment.

## Technologies Used 
- TypeScript
- Cludefare workers
- Wrangler CLI
